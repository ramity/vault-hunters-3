# Setup

- Install docker
- Download files from [here](https://www.curseforge.com/minecraft/modpacks/vault-hunters-1-18-2/files/4516817)
- Unzip into the data directory
- Run the following command:

```
docker compose up -d
```

# Backing up

Execute from root of repo on the host PC:

```
./scripts/backup.sh
```
